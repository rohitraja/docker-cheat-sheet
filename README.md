# docker-cheat-sheet
Docker basic commands

### Docker images
```

$ docker pull <images name>:<tag name>
$ docker images
$ docker images -q
$ docker images -f "dangling=false" -q
$ docker run <image>
$ docker run -it <image>
$ docker rmi -f <image>
$ dockder inspect <image>
```
